Format attendu

Votre fichier Excel doit contenir les colonnes suivantes :

Colonne Requis
Nom Oui
Prénom Oui
Âge Non
Niveau Oui
Acuité Visuelle OD Non
Acuité Visuelle OG Non
Important :
  • La première ligne doit contenir les en-têtes
  • Ville et École = École Active
  • Niveau = Classe (CP, CE1, CE2...)
Conseils
  • Activez d'abord une école dans "École Active"
  • Téléchargez le modèle Excel pour voir un exemple
  • L'acuité visuelle peut être laissée vide
  • Les doublons (même nom + prénom) sont ignorés
  • Les élèves importés auront le statut "Pré-listé"
